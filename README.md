# networking-scanning-projects

 # Description
  This project focuses on network discovery and scanning techniques used to identify active hosts, open ports, and running services within a network. It demonstrates practical applications of networking concepts and basic cybersecurity reconnaissance methods.

The goal of this project is to build a deeper understanding of how devices communicate on a network and how to assess network exposure.

# objectives
Understand core networking concepts (TCP/IP, ports, protocols)
Perform host discovery within a local network
Identify open ports and associated services
Gain hands-on experience with network scanning tools
Analyze potential security risks in a network environment

# Tools i used
Nmap – Network discovery and port scanning
Wireshark – Packet analysis and traffic inspections
Bash/Shell Scripting – Automation of scanning tasks
# Methodology
Network Discovery
Identified active devices using ping sweeps and ARP scans

Port Scanning
Scanned target hosts to detect open, closed, and filtered ports

Service Enumeration
Determined services running on open ports (e.g., HTTP, SSH, FTP)

Traffic Analysis
Captured and analyzed packets to understand communication patterns

# Results
Successfully identified multiple active hosts within the network
Detected open ports and associated services
Gained insights into network structure and communication patterns
Highlighted potential security exposure points
